Kegiatan

Kurang lengkap bilamana sebuah komunitas tanpa kegiatan, demikian pula dengan JUG Indonesia. JUG Indonesia telah lebih dari satu dekade ini mengadakan kegiatan, dimulai dari JaMU (Java Meet Up), dilanjutkan dengan program kegiatan lainnya seperti dengan Oracle untuk Oracle Gelatika, dan mendukung program berhubungan dengan Java seperti Eclipse DemoCamp. JUG Indonesia juga peduli dengan situasi bangsa dan negara, juga melahirkan program kegiatan berkala bernama JUGragan, sebuah kegiatan komunitas untuk meningkatkan jiwa kewirausahaan dan tentu saja semua menggunakan Java. Jangan biarkan diri anda tidak terupdate dengan tidak mendukung program kegiatan ini. Mari menjadi dari bagian dari mekanisme kontribusi JUG Indonesia.

JaMU

JaMU (Java Meet Up) adalah kegiatan berkala JUG Indonesia. Merupakan kegiatan sharing session dari anggota untuk anggota. Bilamana Anda ingin menjadi pembicara atau memiliki topik yang mau dishare di acara ini, silahkan email saja idenya di milis atau menghubungi moderator untuk informasi lebih lanjut.

JUGragan

JUGragan adalah kegiatan sharing yang bertujuan menciptakan para pengusaha baru yang menggunakan teknologi Java baik sebagai komponen, aplikasi atau platformnya. Program ini diciptakan guna mendukung dan meningkatkan jiwa kewirausahaan di Indonesia yang kita ketahui sangat minim.

Oracle Gelatika

Oracle Gelatika adalah sebuah program kerja sama JUG Indonesia dengan Oracle Indonesia. Dengan program utama melakukan sharing session seputar teknologi Java dilingkungan Oracle, yaitu area Fusion Middleware.

Eclipse DemoCamp

Bekerja sama dengan Eclipse dan Meruvian, JUG mengadakan kegiatan berkala Eclipse DemoCamp.
